### ðŸ” Example 2: Find the Largest of 3 Numbers

**Steps**:

1. Take 3 numbers as input
2. Compare them using `if` conditions
3. Print the largest

**Python Code**:

```python
a = 10
b = 25
c = 15

if a >= b and a >= c:
    print("Largest:", a)
elif b >= a and b >= c:
    print("Largest:", b)
else:
    print("Largest:", c)
```
